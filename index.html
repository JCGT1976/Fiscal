📐 Tabla de INPC (para actualización fiscal)

Serie histórica del Índice Nacional de Precios al Consumidor (INEGI, base 2ª quincena julio 2018 = 100) precargada en Supabase desde enero 2020. Se usa automáticamente para calcular el factor de actualización del art. 17-A del CFF. Si el INPC de un mes nuevo aún no aparece (INEGI lo publica en el DOF entre el día 8 y 10 de cada mes), puedes cargarlo con el Excel de abajo o capturarlo cuando el sistema te lo pida al calcular. Formato: Periodo (AAAA-MM) e INPC (número).

🔄 Actualizar impuestos de CxC Fiscal a la fecha actual

Selecciona uno, varios o todos los conceptos de CxC Fiscal, agrupados como prefieras. El sistema calcula actualización (INPC) + recargos según la ley que le aplica a cada tipo de impuesto (CFF para ISR/IVA, LSS para IMSS, Ley del INFONAVIT, o Código Fiscal del Estado de BC para ISN), y simula el efecto de una condonación según lo que cada ley permite.
⚠️ Esto es una herramienta de apoyo interno, no una determinación oficial. Antes de pagar o presentar cualquier trámite de reducción/condonación, valídalo con el fiscalista de MI Global.
